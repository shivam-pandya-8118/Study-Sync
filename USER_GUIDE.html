üöÄ Getting Started

First Time Setup

  1. Open Study Sync: Launch the application in your web browser.
  2. Review Default Subjects: The app comes pre-loaded with common engineering subjects. You can modify these in the "Edit Subjects" tab.
  3. Review Work Types: Default work types like "Assignment", "Lab Manual", "Project Report" are already available.
  4. Add Your First Work Item: Click on "Add Work" tab to create your first assignment or project.

üìù How to Add Work

  1. Navigate to Add Work: Click on the "Add Work" tab.
  2. Select Subject: Choose from the dropdown list of available subjects.
  3. Choose Work Type: Select the type of work (Assignment, Project, Lab Manual, etc.).
  4. Describe the Work: Enter a detailed description of what needs to be done.
  5. Set Issue Date: When the work was assigned (defaults to today).
  6. Set Deadline: The due date for the work.
  7. Enter Faculty Name: Name of the professor or instructor who assigned the work.
  8. Student Name: Your name (pre-filled with "Shivam Pandya").
  9. Submit: Click "Add Work" to save the item.

Pro Tip:

Be specific in your work description. Instead of "Math Assignment", write "Calculus Assignment - Chapter 5 Problems 1-20".

üìà Managing Work Status

Status Types

Not Started
Pending
Completed

How to Update Status

  1. Go to Dashboard: All your work items are displayed here.
  2. Find Your Work: Use search or scroll to find the specific work item.
  3. Click Status Button: Click the appropriate status button (Not Started, Pending, or Completed).
  4. Confirm Change: A popup will ask for confirmation - click "Confirm" to update.

Status Guide:

Not Started: You haven't begun working on it yet.
Pending: You've started but haven't finished.
Completed: The work is finished and ready for submission.

üîç Search and Filter

Search Functionality

Use the search bar to quickly find specific work items by:

  • Subject name
  • Work type
  • Work description
  • Faculty name

Filter Options

Use filter buttons to view work items by status:

  • All: Shows all work items
  • Not Started: Shows only items you haven't started
  • Pending: Shows work in progress
  • Completed: Shows finished work

‚è∞ Understanding Deadlines

Days Left Display

The "Days Left" column shows:

  • Green numbers: Days remaining until deadline
  • Orange "Tomorrow": Due tomorrow
  • Orange "Due Today": Due today
  • Red "Overdue": Past the deadline

Important:

Work items are automatically sorted by deadline - the most urgent items appear at the top of the list.

‚öôÔ∏è Customizing Subjects and Work Types

Adding New Subjects

  1. Go to Edit Subjects: Click the "Edit Subjects" tab.
  2. Enter Details: Fill in subject name and code.
  3. Add Subject: Click "Add Subject" button.

Adding New Work Types

  1. Go to Edit Work Type: Click the "Edit Work Type" tab.
  2. Enter Name: Type the new work type name.
  3. Add Type: Click "Add Work Type" button.

Customization Tips:

You can edit or delete existing subjects and work types, but be careful - deleting items that are used in existing work items will prevent deletion.

‚ú® Best Practices

  1. Regular Updates: Update your work status regularly to maintain accuracy.
  2. Detailed Descriptions: Write clear, specific descriptions for each work item.
  3. Set Realistic Deadlines: Enter accurate deadline dates as provided by your instructors.
  4. Review Daily: Check your dashboard daily to stay on top of upcoming deadlines.
  5. Use Filters: Use status filters to focus on what needs attention.
  6. Keep It Updated: Remove or mark as completed any work that's finished.

üîß Troubleshooting

Common Issues

Can't Delete Subject/Work Type

This happens when the item is being used in existing work items. Complete or delete the related work items first.

Lost Data

Data is stored in your browser's local storage. Clearing browser data will remove all information.

Status Not Updating

Make sure to click "Confirm" in the popup dialog when changing status.
